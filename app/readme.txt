Practica Pokeapi - Documentación de instalación y despliegue

1. Requisitos previos:
   - Android Studio instalado.
   - JDK instalado.
   - Un dispositivo Android o un emulador para probar la aplicación.

2. Instalación:
   - Clona el repositorio de GitHub en tu máquina local usando Git.
   - Abre Android Studio.
   - Selecciona 'Open an existing Android Studio project'.
   - Navega hasta el directorio donde clonaste el proyecto y haz clic en 'Open'.
   - Android Studio debería importar el proyecto y resolver todas las dependencias.

3. Despliegue:
   - Conecta tu dispositivo Android a tu máquina o inicia tu emulador.
   - En Android Studio, selecciona 'Run' -> 'Run 'app''.
   - Selecciona el dispositivo o emulador en el que deseas instalar la aplicación.
   - Android Studio instalará la aplicación en tu dispositivo o emulador y la iniciará.

4. Uso:
   - Al iniciar la aplicación, verás dos botones: 'Iniciar Lista' y 'Pokemon Aleatorio'.
   - Al pulsar 'Iniciar Lista', la aplicación mostrará una lista de Pokémon en la consola.
   - Al pulsar 'Pokemon Aleatorio', la aplicación mostrará un Pokémon aleatorio en la consola.
